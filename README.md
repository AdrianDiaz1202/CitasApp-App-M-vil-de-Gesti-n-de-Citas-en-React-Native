## CitasApp – Aplicación Móvil con React Native

Aplicación móvil desarrollada con React Native + TypeScript, enfocada en la gestión básica de citas.
Este proyecto forma parte de mi proceso de formación como desarrollador de software y representa mi práctica en desarrollo móvil multiplataforma.

- Demo del Proyecto

- Proyecto ejecutable en Android e iOS mediante React Native CLI.

## Tecnologías Utilizadas:

- React Native 0.82

- React 19

- TypeScript

- Metro Bundler

- React Native Date Picker

- React Native Safe Area Context

- Jest (Testing)

- Node.js >= 20

## Características Principales:

- Selección de fechas mediante Date Picker

- Aplicación multiplataforma (Android / iOS)

- Configuración moderna con TypeScript

- Estructura base escalable

- Configuración de pruebas con Jest

- Buenas prácticas en estructura de proyecto

- Arquitectura del Proyecto

## Instalación y Ejecución:

1️⃣ Clonar repositorio:

git clone https://github.com/TU-USUARIO/TU-REPO.git

2️⃣ Instalar dependencias:

npm install

3️⃣ Ejecutar en Android:

npm run android

4️⃣ Iniciar servidor Metro:

npm start
